# SysAdmin Core Lab

This repository documents a hands-on systems administration lab designed to build real-world infrastructure skills.

## Goals
- Build and manage a Windows domain environment
- Understand DNS, DHCP, and IP networking
- Administer Linux services and permissions
- Practice troubleshooting and documentation

## Lab Environment
- Host OS: Windows 11
- Hypervisor: VirtualBox
- Windows Server 2022 (Domain Controller)
- Windows 10/11 Client
- Ubuntu Server (no GUI)
